# Secure Hospital Imaging - Frontend

React-based doctor console for the Secure Hospital Imaging System.

## Setup

```bash
npm install
npm start
```

The app runs at `http://localhost:3000` and expects the backend at `http://localhost:5000`.

## Pages

- Login / Signup
- Dashboard (upload + demo flow)
- Patient Records
- Tamper Simulation & Analysis

## Notes

- The UI highlights that original scans are never stored.
- Protected images can be downloaded from patient records.
